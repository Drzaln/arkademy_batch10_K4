## Soal 1

REST API (Representational State Transfer Application Programming Interface) adalah style yang digunakan dalam pembuatan web service. JSON (JavaScript Object Notation) adalah notasi penulisan data yang sangat mudah dibaca oleh manusia, JSON memberikan kemudahan penanganan data tanpa peduli bahasa apa yang digunakan dalam aplikasi tersebut.
<br>Stack yang digunakan yaitu Dart dengan fungsi Map toJson untuk mengubah array ke bentuk JSON, parameter bisa diubah didalam class main.</br>
